# Stock Photo License Inventory

Список всех стоковых изображений в макете Figma, для которых потребуется купить
лицензию перед запуском сайта в продакшен.

Заполняется **по мере прохождения каждой страницы**. На каждой странице услуги
обычно 2-3 стоковых фото + фото в общем футере (если оно тоже стоковое).

## Как опознать сток

Имя файла в Figma сохраняется от исходника. Индикаторы:
- `istockphoto-XXXXXXX` → iStock by Getty
- `premium_photo-XXXXXX` → Unsplash+ (платная подписка)
- `photo-XXXXX` → Unsplash (часто бесплатно при атрибуции)
- `shutterstock_XXXXXX` → Shutterstock
- `AdobeStock_XXXXXX` → Adobe Stock
- `pexels-XXXXX` → Pexels (бесплатно)

## Что делать перед продакшеном

1. Открыть исходный сервис, найти фото по ID (например iStock id=1319189368
   → https://www.istockphoto.com/photo/1319189368)
2. Проверить лицензию (Standard / Extended) и стоимость
3. Купить — обычно $12-$200 в зависимости от размера и типа лицензии
4. Заменить файл в проекте на купленную полноразмерную версию
5. Сохранить документ-лицензию в облако клиента

## Легенда статусов
- 🟡 **placeholder** — обнаружен в макете, лицензия не куплена, в проекте используется превью с водяным знаком
- 🟢 **licensed** — лицензия куплена, в проекте лежит финальная версия
- 🔴 **replace** — клиент решил заменить фото
- ⚪ **not-stock** — оказалось не стоковым (фото клиники, команды и т.п.)
